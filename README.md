# TinyMCE Prototype

https://www.tiny.cloud/

This project was created to test the prototype free version of TinyMCE's rich text editor. I added in the following plugins: 

### Bulleted list
### Autolinks
### Autosave
### Images
### Charmap
### Anchor
### Search Replace
### Visual Blocks
### Code
### Code Toolbar
### Fullscreen
### Insert Date/Time
### Media
### Table
### Preview
### Help button
### Wordcount 
### Emoticons
### Code Sample
### Quickbars 

## Available Scripts

In the project directory, you can run:

### `npm start`
### `npm test`
### `npm run build`
